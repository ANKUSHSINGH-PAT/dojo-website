# dojo-website

## Development

1. `npm install`
2. `bower install`
3. `grunt develop`
4. Visit http://localhost:1337/


## Building Reference Guide

### Requirements

* Sphinx

1. `sudo easy_install pip`
2. `sudo pip install sphinx`
3. `grunt refguide`



## Building API Docs

1. `cd src/documentation/api`
2. `node spider.js`
