dojox/gfx/_gfxBidiSupport.defaultText (version 1.10)

Defines the default Text prototype.

See thedojox/gfx/_gfxBidiSupport.defaultText reference documentation for more information.

Property Summary

  • align
    • The horizontal text alignment, one of &#39;start&#39;, &#39;end&#39;, &#39;center&#39;.
  • decoration
    • The text decoration , one of &#39;none&#39;, ... .
  • kerning
    • Whether kerning is used on the text, boolean default value true.
  • rotated
    • Whether the text is rotated, boolean default value false.
  • text
    • The text to be displayed, default value empty string.
  • type
    • Specifies this is a Text shape, value &#39;text&#39;.
  • x
    • The X coordinate of the text position, default value 0.
  • y
    • The Y coordinate of the text position, default value 0.
