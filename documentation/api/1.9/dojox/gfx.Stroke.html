dojox/gfx.Stroke (version 1.9)

A stroke defines stylistic properties that are used when drawing a path.

Property Summary

  • cap
    • The endcap style of the path.
  • color
    • The color of the stroke, default value &#39;black&#39;.
  • join
    • The join style to use when combining path segments.
  • style
    • The style of the stroke, one of &#39;solid&#39;, ... .
  • width
    • The width of a stroke, default value 1.
