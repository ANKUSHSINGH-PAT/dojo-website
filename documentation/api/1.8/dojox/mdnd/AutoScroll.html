dojox/mdnd/AutoScroll (version 1.8)

Activate scrolling while dragging a widget.

Usage:

var foo = new AutoScroll();
Defined by dojox/mdnd/AutoScroll

Property Summary

  • interval
    • default mouse move offset
  • marginMouse

  • recursiveTimer

Method Summary

  • _autoScrollDown() Manage the down autoscroll.
  • _autoScrollLeft(e) Manage the left autoscroll.
  • _autoScrollRight() Manage the right autoscroll.
  • _autoScrollUp() Manage the up autoscroll.
  • checkAutoScroll(e) Check if an autoScroll have to be launched.
  • destroy()

  • init()

  • setAutoScrollNode(node) set the node which is dragged
  • stopAutoScroll() Stop the autoscroll.
