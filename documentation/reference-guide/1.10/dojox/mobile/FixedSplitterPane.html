Introduction¶

Deprecated. Since V1.8, use dojox/mobile/Container instead.

FixedSplitterPane is a pane widget that is used in a dojox/mobile/FixedSplitter. It is a widget, but can be regarded as a simple <div> element.

Usage¶

dojox/mobile/FixedSplitter has a dependency on this widget. You should require dojox/mobile/FixedSplitter rather than dojox/mobile/FixedSplitterPane.

Constructor Parameters¶

None.
