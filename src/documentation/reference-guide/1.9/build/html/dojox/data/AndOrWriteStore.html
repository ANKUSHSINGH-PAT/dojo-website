dojox.data.AndOrWriteStore¶

Author:Frank Fortson
Since:1.2

Contents

  • dojox.data.AndOrWriteStore
    • API Support

dojox.data.AndOrWriteStore is a clone of dojo.data.ItemFileWriteStore. The only significant difference is that like AndOrReadStore, AndOrWriteStore supports more complex queries. Please refer to dojo.data.ItemFileWriteStore for general information about write support and refer to dojox.data.AndOrReadStore for the query enhancements.
