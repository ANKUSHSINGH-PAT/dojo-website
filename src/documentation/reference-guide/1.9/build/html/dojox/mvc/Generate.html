dojox.mvc.Generate¶

Authors:Rahul Akolkar, Ed Chatelain
Developers:Rahul Akolkar, Ed Chatelain
since:V1.7

Contents

  • dojox.mvc.Generate
    • Parameters
    • Examples

A container that generates a view based on the data model its bound to. A generate introspects its data binding and creates a view contained in it that allows displaying the bound data. Child dijits or custom view components inside it inherit their parent data binding context from it.

Parameters¶

Parameter Type Default Description
widgetMapping Object {“String” : “dijit.form.TextBox”} The mapping of types to a widget class. ie: {‘String’ : ‘dojox.mobile.TextBox’}
classMapping Object {“Label” : “generate-label-cell”, “String” : “generate-dijit-cell”, “Heading” : “generate-heading”, “Row” : “row”} The mapping of class to use for each node.
idNameMapping Object {“String” : “textbox_t”} The mapping of id and name to use. Set idNameMapping to override this. A count will be added to the id and name.
