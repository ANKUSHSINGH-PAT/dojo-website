dojox.mvc.Output¶

Authors:Rahul Akolkar, Ed Chatelain
Developers:Rahul Akolkar, Ed Chatelain
since:V1.7

Contents

  • dojox.mvc.Output
    • Parameters
    • Available Methods
    • Examples

A simple widget that displays templated output, parts of which may be data-bound.

Parameters¶

Parameter Type Default Description
ref String or StatefulModel   Deprecated. ref is only used with StatefulModel which is Deprecated, use value: at(model, ‘attr’). The value of the data binding expression passed declaratively by the developer. This usually references a location within an existing datamodel and may be a relative reference based on the parent / container data binding (dot-separated string).
exprchar Character ‘$’ Character to use for a substitution expression, for a substitution string like ${this.value}

Available Methods¶

  • dojox/mvc/Output.set

Override and refresh output on value change.
